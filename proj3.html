App de marcações para barbearia

Aplicação para agendamento de serviços de barbearia, com funcionalidades de login, gestão de reservas e notificações

Tecnologias Utilizadas

  • Kotlin
  • Firebase
  • Android

Ecrã de Login

  • Permite o login de utilizadores através de email e palavra-passe.
  • Utiliza a autenticação do Firebase.
  • Inclui um botão para navegar para o ecrã de registo.
  • Contém um login especial para o administrador.

Ecrã de Registo

  • Permite que novos utilizadores criem uma conta.
  • Valida a confirmação da palavra-passe e cria um novo utilizador no Firebase Auth.
  • Guarda a informação do utilizador (nome e telefone) no Firestore.
  • Inclui um botão para regressar ao ecrã de login.

Ecrã da Página Inicial

  • Página de entrada para o utilizador, exibindo a última marcação agendada.
  • Exibe galerias de imagens de clientes e do espaço da barbearia.
  • Permite navegar para outras secções da aplicação através do menu lateral.

Meu perfil

  • Ecrã para o utilizador visualizar e gerir as suas marcações.
  • Possui três separadores: "Perfil", "Marcações" e "Saldo".
  • Obtém e exibe a informação do utilizador e das marcações do Firestore.

Ecrã de Código de Recomendação

  • Exibe um código de recomendação para o utilizador.
  • Inclui um botão para partilhar o código.
  • Permite o utilizador voltar à página inicial.

Ecrã de Marcações

  • Exibe uma lista de serviços disponíveis, obtidos do Firestore.
  • Permite ao utilizador selecionar um serviço para continuar o processo de marcação.
  • Contém a barra de navegação inferior padrão para alternar entre ecrãs.

Ecrã de Barbeiros

  • Apresenta uma lista dos barbeiros disponíveis.
  • Permite ao utilizador selecionar um barbeiro para a sua marcação.
  • Obtém a lista de barbeiros a partir do Firestore.

Ecrã de Seleção de Data e Hora

  • Permite ao utilizador escolher a data da sua marcação usando um seletor de data.
  • Apresenta uma lista de horários disponíveis.
  • Navega para o próximo ecrã após a seleção da data e hora.

Ecrã de Confirmação de Marcação

  • Exibe o resumo da marcação selecionada pelo utilizador.
  • Permite ao utilizador guardar a marcação no Firestore.
  • Mostra o serviço, barbeiro, data e hora selecionados.

Admin Home Screen

  • Ecrã de administração para gerir serviços, barbeiros, imagens de portfólio e marcações.
  • Permite adicionar, editar e apagar serviços e barbeiros.
  • Oferece a possibilidade de visualizar e editar marcações existentes.
  • Integra-se com o Firestore para persistência de dados.
