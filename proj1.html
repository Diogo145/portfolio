Website Estaleiro de Sarilhos Pequenos

Website institucional desenvolvido para o Estaleiro de Sarilhos Pequenos, apresentando informações sobre serviços, história, projetos, entre outros, com design responsivo e otimizado para diferentes dispositivos.

Tecnologias Utilizadas

  • WordPress
  • PHP
  • CSS3
  • HTML5
  • JavaScript

Ferramenta / Metodologia

  • Docker: Plataforma que permite criar, distribuir e executar aplicações em containers, garantindo que funcionam de forma consistente em diferentes ambientes. Explicar muito mais
  • MobaXTerm, VS, Extensões
  • WordPress como base de conteúdo
  • PHP, HTML5, CSS3 e JavaScript
  • Three.js e Visody para conteúdos 3D interativos
  • Plugins: Visody, Gtranslate, WP SMTP, MaxUploader

Gtranslate

  • Tradução automática do conteúdo do site
  • Cobertura linguística ampla
  • Facilidade de alternância entre idiomas
  • Contribuição para a internacionalização do site
  • Desvantagem: Qualidade da tradução

Visody

  • Integração e visualização direta de modelos 3D no WordPress
  • Suporte a ficheiros 3D interativos (.glb, .gltf)
  • Manipulação, rotação e ampliação dos modelos em tempo real
  • Experiência imersiva para os utilizadores
  • Compatibilidade com dispositivos móveis
  • Desempenho fluido na visualização dos modelos
  • Ideal para apresentar modelos 3D de embarcações tradicionais
  • Plugin MaxUploader

WP SMTP

  • Envio confiável de e-mails pelo site
  • Redução de problemas com spam ou bloqueios
  • Utilização de credenciais SMTP personalizadas
  • Substituição do envio nativo de e-mails do WordPress
  • Garantia de que os e-mails chegam corretamente ao destino

Homepage

  • Página inicial com visão geral do site
  • Destaque para blogs e projetos
  • Construída com o template front-page.php
  • Usa componentes como project-section.php e blog-section.php
  • Conteúdos exibidos dinamicamente e sempre atualizados

Secção de Blogs da Homepage

  • blog-section.php realiza consulta personalizada ao tipo de conteúdo post
  • Exibe os artigos mais recentes do blog em formato de carrossel interativo
  • Uso do Owl-Carousel (lib JS)
  • Apresenta miniaturas, títulos e resumos automáticos para cada publicação

Secção de Projetos da Homepage

  • projetos-section.php faz uma consulta personalizada (WP_Query) ao tipo de conteúdo projeto
  • Exibe até 8 projetos recentes em formato de carrossel interativo
  • Cada item usa o estilo project-card, com imagem, excerto e link para o conteúdo completo

Página dos Serviços

  • Página de serviços construída com blocos manuais no editor do WordPress, sem uso de templates PHP
  • Estrutura organizada com blocos de Grupo, títulos e parágrafos com classes CSS personalizadas
  • Estilo visual definido no ficheiro CSS do tema, garantindo uniformidade e responsividade

Página dos Blogs

  • Página de blog utiliza o template index.php e carrega os posts (post type) dinamicamente
  • Cada blog é renderizado com o componente project-card.php

Página do Blog Específico

  • Botão "Ver Mais" direciona para a página individual do blog, gerada com o template universal single.php
  • O template chama o ficheiro content-post.php para exibir o conteúdo completo
  • Mostra título, texto detalhado e imagens, mantendo coerência visual e estrutural

Página dos Projetos

  • Criação de um tipo de post: Projeto
  • Página de projetos usa o template archive-projeto.php e carrega os projetos (project type) dinamicamente
  • Cada projeto é exibido com o componente project-card.php
  • Exibe imagem, título (no-translate), excerto e botão

Página do Projeto Específico

  • Botão “Ver Mais” direciona para a página individual do projeto, com o template single.php
  • Esse template invoca o ficheiro content-projeto.php para exibir o conteúdo completo
  • Apresenta título (no-translate), texto, imagens e modelo 3D interativo (VISODY)
  • Limite de carregamento por ficheiro (Max_Uploader)

Secção do Formulário da Página dos Contactos

  • Formulário de contacto com campos personalizados e com validação:
    • Sanitização
    • reCAPTCHA
  • Armazenamento dos dados na tabela personalizada wp_contact_us_pedidos
  • Envio de duas mensagens em HTML (para o admin e para o utilizador) e mensagens de feedback claras em caso de erro ou sucesso (WP SMTP)

Secção da Morada da Página dos Contactos

  • Informações de contacto e morada
  • Email e telemóvel a um clique
  • Uso de Google Maps com iframe
  • Morada - no-translate
  • Complementa o formulário de contacto, facilitando a localização e interação rápida com o estaleiro

Página da Galeria do Rio Tejo

  • Uso do template personalizado page-gallery.php
  • Uso do Glightbox (lib JS) para visualização ampliada
  • Navegação intuitiva e visual moderno, com experiência fluida em desktop e mobile

Página da Galeria do Estaleiro

  • Utiliza o mesmo template do page-gallery.php
  • Herda a estrutura e estilo do template original
  • Permite consistência visual entre galerias diferentes
  • Facilita manutenção e atualização do layout
  • Compatível com Glightbox e navegação responsiva

Secção Sobre Nós - História do Estaleiro

  • Identidade
  • Missão e valores do Estaleiro Naval de Sarilhos Pequenos
  • Destacar a carpintaria naval artesanal
  • Preservação do património cultural do estuário do Tejo
  • Narração envolvente sobre a história do estaleiro, sua importância comunitária e o saber acumulado de gerações

Secção Sobre Nós - Q&A's ao Mestre Jaime

  • Rigor técnico na construção e restauro de embarcações tradicionais e a dimensão humana da arte
  • A escassez da mão de obra especializada e os desafios para a continuidade do ofício
  • Abertura do estaleiro à comunidade com visitas guiadas e oficinas educativas, valorizando o legado patrimonial material e imaterial

Página de Erro 404

  • Página 404 personalizada utilizando o 404.php
  • Inclui título, subtítulo explicativo, mensagem criativa e formulário de pesquisa para facilitar a navegação
  • Botão para retornar à página inicial

Página Search Results

  • Página com resultados da pesquisa personalizada utilizando o template search.php
  • Cada resultado é exibido com o componente project-card.php
  • Botão para os respetivos conteúdos completos

Header

  • Header utiliza o template header.php
  • Inclui logo, título (no-translate) e imagem
  • Menu principal com itens e sub-itens
  • Menu encolhido em ecrãs pequenos
  • Responsivo

Footer

  • Footer utiliza o template footer.php
  • Menu principal com itens
  • Contactos
  • Dedicatória
  • Pesquisar
  • Responsivo
